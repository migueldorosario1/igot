# @igot/web

App web (PWA) do igot — a interface de leitura com o assistente de IA.

> **Status:** scaffold vazio. O código será escrito na Fase 1 do roadmap.

## Planejado

- Next.js (App Router) + React + TypeScript
- PWA instalável (leitura offline)
- Layout: área de leitura à esquerda + painel do assistente à direita
- Upload de EPUB/PDF → renderização
- Seleção de texto → menu contextual (Traduzir / Explicar / Resumir)
- Conexão com o backend de IA (Z.ai/GLM no MVP)
