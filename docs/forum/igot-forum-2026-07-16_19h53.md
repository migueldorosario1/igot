## ✅ Problemas encontrados e soluções

### 1. Título do capítulo duplicado (EPUB)
- **Problema**: o título aparecia duas vezes — no header E no conteúdo
- **Causa**: `chapter?.title` renderizado tanto no `<h2>` do conteúdo quanto no header
- **Solução**: removido do conteúdo (já está no header)
- **Arquivo**: `Reader.tsx`

### 2. Livros antigos não abriam mais (quebra de migração)
- **Problema**: livros salvos antes da estante (store `sessions`, chave `current`) não apareciam
- **Causa**: a estante nova busca no store `books` (IndexedDB v2); o legado ficou isolado
- **Solução**: `migrateLegacyBook()` roda automaticamente quando a estante carrega — detecta o livro antigo, transfere pro novo store com ID único, limpa o legado. Deduplica por fileName+fileSize.
- **Arquivo**: `db.ts`, `page.tsx` (home)

### 3. Livros duplicados na estante
- **Problema**: reabrir o mesmo arquivo criava um novo entry na estante
- **Causa**: cada upload gerava ID novo sem checar se já existia
- **Solução**: `handleFile` agora busca por `fileName + fileSize` antes de gerar ID novo; se existe, reusa
- **Arquivo**: `page.tsx` (home), `session.ts`

### 4. Zoom invertido (aumenta zoom → letra diminuía)
- **Problema**: ao aumentar o zoom, a página estreitava em vez de crescer
- **Causa**: feedback loop — o `parent.clientHeight` mudava a cada render porque o canvas crescia, mudando o `fit`, que recalculava o zoom
- **Solução**: medição agora pega o **avô** do container (`.reader-scroll`), que tem altura fixa pelo flex/grid
- **Arquivo**: `PdfPageCanvas.tsx`

### 5. Quadrado preto embaixo da página
- **Problema**: área preta visível abaixo do canvas do PDF
- **Causa**: quando o canvas era menor que 100% da altura (zoom reduzido), o espaço extra mostrava o fundo do body (preto em dark mode)
- **Solução**: `.pdf-page-container` agora tem `background: var(--bg)` (acompanha o tema). `.pdf-page-wrapper` e `.pdf-canvas` sempre `background: white !important`
- **Arquivo**: `globals.css`

### 6. Scroll não funcionava no PDF (página não descia)
- **Problema**: não dava pra rolar a página pra baixo pra ver o resto do PDF
- **Causa**: `touch-action: pan-y` no `.reader-scroll` bloqueava o scroll vertical natural em iOS/Safari quando o conteúdo era mais alto que a área visível
- **Solução**: trocado pra `touch-action: auto` (permite scroll vertical normal)
- **Arquivo**: `globals.css`

### 7. Erro de sintaxe que travou o build (2 vezes)
- **Problema 1**: `});` em vez de `};` no `deleteBookFromLibrary` (db.ts)
- **Problema 2**: `id: bookId,` duplicado no `handleFile` (page.tsx)
- **Solução**: corrigidos
- **Arquivos**: `db.ts`, `page.tsx`

### 8. Ícones pequenos demais no header
- **Problema**: botões do topbar eram difíceis de tocar (36px em vez de 44px)
- **Solução**: `.gear` agora 44x44px (mínimo Apple HIG pra touch), `.brand` 20px, com hover scale
- **Arquivo**: `globals.css`

## ✅ Features implementadas nesta sessão

### 1. Estante (biblioteca de múltiplos livros)
- Home (`/`) agora é um **grid de capas** dos livros
- Cada livro: capa (ou placeholder 📖), título, autor, formato, progresso
- Botão **"+ Adicionar livro"**
- Botão **"🗑 Limpar tudo"** (limpa estante inteira)
- Botão **🗑** em cada livro (excluir individual, aparece no hover)
- Quando vazia: mostra o onboarding (Uploader redesenhado)

### 2. Rotas com URL própria
- `/` → estante
- `/book/[id]` → lendo um livro específico (URL compartilhável)
- Logo **"💡 igot"** clicável → volta pra estante
- Ícone **📚** no topbar do livro → volta pra estante

### 3. Migração de dados (v1 → v2)
- `openDBWithBooks()`: abre DB v2 com store `books` (keyPath=`id`)
- `migrateLegacyBook()`: transfere livro legado automaticamente
- Compatível com o store antigo (`sessions`)

### 4. Redesign visual
- **Onboarding** (Uploader): hero + 3 features + aviso de config + badge privacidade
- **Tipografia serifada** no leitor EPUB (Iowan/Palatino/Georgia)
- **Barra de progresso** no header do Reader (estilo Kindle)
- **Design tokens** (CSS variables pra fontes, tamanhos, espaçamento)
- **Seleção amarela** unificada via variável `--selection-bg` (acompanha dark mode)
- **Painel da IA**: loading com label, resultado renderiza **negrito** markdown

### 5. Barra de navegação rápida
- Slider horizontal no rodapé do leitor
- Arrasta pra pular pra qualquer página
- Botões ‹ › + contador (página X de Y)
- Funciona pra PDF e EPUB

### 6. TopBar organizado
- Esquerda: 💡 igot (logo) + 📚 (estante)
- Direita: login Google + ⚙️ (configurações)
- Ícones 44px (mínimo touch Apple HIG)
